## âš™ï¸ How It Works

1. User selects text content on any webpage.
2. Chrome Extension sends request to **Spring Boot backend** via REST API.
3. Backend processes prompt with **Google Gemini LLM**.
4. AI-generated summary/analysis returned and displayed in extension popup.
5. Users can save insights to Notes for future reference.
